# Enterprise Help Desk Home Lab

## Overview

This project simulates an enterprise IT support environment commonly found in government, defense, and corporate organizations.

The lab focuses on Active Directory administration, incident management, user provisioning, ticket tracking, SLA monitoring, and Tier 1/Tier 2 troubleshooting procedures.

---

## Objectives

- Build a realistic IT support environment
- Practice Active Directory administration
- Simulate Help Desk ticket workflows
- Document troubleshooting procedures
- Track service level agreements (SLAs)
- Develop professional IT documentation
